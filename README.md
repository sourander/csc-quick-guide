## About This Repository

This project serves as a quick start guide for university students in Finland who need to train Convolutional Neural Networks (CNNs) using CSC (Center for Scientific Computing) resources.

It covers:
- Setting up a CSC student project.
- Accessing the **Puhti** supercomputer.
- Running GPU-accelerated jobs using Slurm.
- Training PyTorch models on custom datasets.
